💻 大数据 24-1

基于深度学习的检测船舶分类系统

利用计算机视觉技术，对海面船舶进行实时检测与分类。该项目基于 YOLO 算法模型，能够有效识别多种船舶类型，适用于港口监控与海上交通管理。

👁️ 核心算法：前沿的 YOLOv8 模型

本项目搭载了业界领先的目标检测算法 —— YOLOv8。通过对大规模船舶数据集的迁移学习与微调，模型能够在复杂多变的海面环境下，实现毫秒级延迟的实时目标检测，并对多种船舶类别进行高精度分类。

🧠 交互升级：视觉与知识的融合

实现了从“看清”到“看懂”的跨越。系统集成了智能 API 接口，当视觉模型识别出特定船舶后，用户可即时询问。后台会结合识别结果调用知识库 API，为用户提供关于该船舶种类、用途及相关航海知识的即时科普。

🌐 应用前景：海洋大数据赋能

该系统为海洋监管提供了智能化解决方案。可广泛应用于港口自动化作业监控、海上非法船只排查、航道交通流量统计以及航海科普教育等领域，展现了人工智能在海洋大数据时代的广阔应用潜力。
